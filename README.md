# blinkgtk.org

[blinkgtk.org](https://blinkgtk.org) の公開サイト（GitHub Pages でホスティング）。

BlinkGTK は GTK4 のための Chromium / Blink ベース WebView エンジンです。
このリポジトリは告知ページの静的コンテンツのみを含みます。

## 構成

- `index.html` — トップページ（最小の告知ページ）
- `CNAME` — カスタムドメイン設定（`blinkgtk.org`）

## デプロイ

`main` ブランチへの push で GitHub Pages が自動公開します。
